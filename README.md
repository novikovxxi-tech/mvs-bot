# Бот «МВС СТ АУК №1» — Telegram

Бот для подачи заявок на материал и просмотра списка техники.
Платформа: Telegram · Python + aiogram 3

---

## Структура файлов

```
mvs_bot_tg/
├── bot.py           — основной файл (handlers, FSM)
├── database.py      — работа с SQLite базой данных
├── keyboards.py     — все клавиатуры и кнопки
├── config.py        — настройки, списки техники/материалов/ответственных
├── .env             — токен и ID администраторов (создать из .env.example)
├── .env.example     — пример файла окружения
├── requirements.txt — зависимости Python
└── bot.db           — база данных (создаётся автоматически при первом запуске)
```

## Установка и запуск

### 1. Получить токен бота

1. Откройте Telegram, найдите **@BotFather**
2. Отправьте `/newbot`
3. Введите имя бота: `МВС СТ АУК №1`
4. Введите username (латиницей, заканчивается на `bot`): например `mvs_st_auk_bot`
5. Скопируйте токен вида `7123456789:AAFxxx...`

### 2. Настроить окружение

```bash
# Скопировать файл настроек
cp .env.example .env

# Открыть и вписать токен
nano .env
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить бота

```bash
python bot.py
```

---

## Узнать свой Telegram user_id

Запустите бота и отправьте команду `/id` — бот ответит вашим ID.
Затем добавьте его в `.env` в переменную `INITIAL_ADMIN_IDS` и перезапустите бота.

## Хостинг (чтобы бот работал 24/7)

### Railway.app (рекомендуется, бесплатно)

1. Загрузите папку `mvs_bot_tg` на [GitHub](https://github.com) в новый репозиторий
2. Зайдите на [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub**
3. Выберите репозиторий
4. Зайдите в **Variables** и добавьте:
   - `BOT_TOKEN` = ваш токен
   - `INITIAL_ADMIN_IDS` = ваш Telegram ID
5. Бот запустится автоматически

### Render.com (бесплатно)

1. Загрузите код на GitHub
2. На [render.com](https://render.com) → New → **Background Worker**
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `python bot.py`
5. Добавьте переменные окружения

### VPS (любой, от 200 руб/мес)

```bash
git clone <ваш репозиторий>
cd mvs_bot_tg
pip install -r requirements.txt
cp .env.example .env && nano .env

# Запуск в фоне
nohup python bot.py &

# Или через systemd / screen / tmux
```

## Возможности бота

### Для всех пользователей
- 📦 Подать заявку на материал (с выбором техники)
- 📋 Просмотреть свои заявки и их статусы
- 🚜 Посмотреть список техники (14 единиц)
- ❌ Отозвать заявку со статусом «Новая»

### Для администраторов (дополнительно)
- 📂 Просмотреть все заявки с фильтром по статусу
- 🔄 Менять статус заявок: Новая → В работе → Выдано / Отклонено
- 🔔 Получать уведомления о новых заявках прямо в Telegram

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Запуск / главное меню |
| `/menu` | Главное меню |
| `/заявка` | Новая заявка |
| `/мои_заявки` | Мои заявки |
| `/cancel` | Отменить действие |
| `/id` | Показать свой Telegram ID |
| `/help` | Справка |
